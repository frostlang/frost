frost programming language


## About

forst is a programming language with a focus on low-friction systems programming.

## Example
```

import("io");

@debuggable
Person : type {
    name : String;
    age  : U8;
}

main : pub fn (args : []String){

    person : const Person = {
        args[0],
        args[1].to_u8()
    };

    io.println(person);

}

```
